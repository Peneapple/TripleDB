Curated evidence, explicit provenance

Higher-order genetic interactions, organized as citable records.

TripleDB is designed to integrate manually curated triple- and higher-order gene relationships with stable TDB accessions, source BioGRID records, publication evidence and transparent curation notes.

Designed for first-time users: one search box, visible examples and no login.

Core user journeys

Everything a database visitor should find quickly

The website separates curated biological conclusions from the source-evidence layer, while keeping both searchable and downloadable.

⌕

Search curated interactions

Query by gene symbol, alias, TDB accession, BioGRID interaction ID or publication, then filter by species, relation type and curation origin.

↳

Verify source evidence

Use a BioGRID interaction ID to inspect the preserved source record and understand why multiple pairwise records were aggregated.

⇩

Download a frozen release

Provide the curated table, filtered source-evidence table, schema, checksums, license terms and release notes together.

Maintenance plan

Stable IDs and documented updates

  • TDB accessions are never reassigned.
  • New records append to the existing accession series.
  • Revisions and deprecations remain visible in a change log.
  • Source releases and added record counts are reported per version.

Proposed cadence: Planned at least annually.
