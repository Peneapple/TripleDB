Classification note

How interaction order is counted

  • Triple: records with three indexed genes.
  • Quadruple or higher: records with four or more indexed genes, supported by Gene_A through Gene_G in the core table.
  • Pair-context / lower-order: a small set of records retained because they are directly relevant to higher-order interpretation.
Automatic updates

Replace files, keep paths stable

When a new release is ready, replace the fixed-name JSON files in data/. The statistics above are then recalculated from each JSON file's _meta section when served through a local web server or GitHub Pages.

Opening the site directly with file:// shows the bundled snapshot because browsers commonly block local JSON fetching.
